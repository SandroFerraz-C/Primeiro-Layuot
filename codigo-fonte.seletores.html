< h1> Seletores CSS < /h1>

< p> Um documento HTML bem estruturado fornece a estrutra a qual seus estilos são aplicados. Para poder estilizar um determinado elemento HTML utilizando CSS, você precisa ter uma maneira de selecionar esses elementos. Na CSS a parte de uma regra de estilo que faz isso e chamado de < b> Seletor < /b> < /p>

< h2> Seletor tipo (tags)

< p> É o seletor mais abrangente da CSS, ele casa com todos os elementos encontrados. < br> Exemplo:

< section class="box-1">

< p> < b> h1 < /b> < p class="f">{ < span class="font01"> Font-size: < /span> 20px;} < /p> < br> < br>

< b> P < /b> < p class="f"> { < span class="font01"> Font-size: < /span> 10px;}



< /section>

< p>Os estilos acima irão refletir em todos os elementos "p" e "h1" da pagina < /p>

< h2> Agrupamento de seletores < /h2>

< p> Para aplicar uma regra CSS comum a vários seletores você pode agrupa-los. < br> < br> Abaixo vemos algumas regras em comum. < /p>

< section class="box-1"> < p> < b>h1 < /b> { < span class="font01"> color: < /span> red;} < br> < b>h2 < /b> { < span class="font01"> color: red;} < br> < b> p < /b> { < span class="font01"> color: < /span> red;} < /p>

< /section>

< p> Aqui eles foram agrupados. < /p>

< section class="box-1">

< p> < b> h1, h2, p < /b> { < span class="font01"> color: < /span> red;} < /p>

< /section> < br> < br>

< h2> Seletor de classe < /h2>

< p> podemos aplicar o seletor ao atributo "Class" de qualquer elemento HTML. Se a intenção é aplicar a mesma regra CSS a diferentes elementos, você pode utilizar um < b> Seletor de Classe < /b>, segue exemplo: < /p>

< section class="box-1">

< p> < p < span class="font02"> class < /span> = < span class="font03"> "diferente" < /span> Este parágrafo sairá diferente por causa da utilização < /p>

< /section>

< p> A CSS deve começar com um "." (ponto) imediatamente seguido pelo nome da classe, veja: < /p>

< section class="box-1">

< p> .diferente { < span class="font01"> color < /span>: black;} < /p>

< /section>

< p> O uso do seletor de classe possibilita estilos diferentes para o mesmo tipo de elemento, exemplo: < b>

< section class="box-1">

p < /b> { < br> < span class="font01"> margin: < /span> < span class="font02"> 10px < /span>; < br>} < br> < b> p < /b> .destaque { < br> < span class="font01">font-weight: < /span> bold; < br>} < br> < b> p < /b> .discreto { < br> color: gray; < br> } < /p>

< /section>

< p> É permitido declar mais de um nome para a classe do mesmo elemento separando os nomes por espaço, exemplo: < br>

< section class="box-1">

< p < span class="font02"> class < /span> < span class="font03"> "destaque discreto" < /span> > Este parágrafo sofrerá estilização de duas regras CSS < /p>

< /section>

< p> O identificador id pode ser aplicado na maioria dos elementos HTML, só que ele deve ser unico em sua página. < br>
