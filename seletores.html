Seletores CSS

Um documento HTML bem estruturado fornece a estrutra a qual seus estilos são aplicados. Para poder estilizar um determinado elemento HTML utilizando CSS, você precisa ter uma maneira de selecionar esses elementos. Na CSS a parte de uma regra de estilo que faz isso e chamado de Seletor

Seletor tipo (tags)

É o seletor mais abrangente da CSS, ele casa com todos os elementos encontrados.
Exemplo:

h1

{ Font-size: 20px;}



P

{ Font-size: 10px;}



Os estilos acima irão refletir em todos os elementos "p" e "h1" da pagina



Agrupamento de seletores

Para aplicar uma regra CSS comum a vários seletores você pode agrupa-los.

Abaixo vemos algumas regras em comum.

h1 { color: red;}
h2 { color: red;}
p { color: red;}

Aqui eles foram agrupados.

h1, h2, p{ color: red;}



Seletor de classe

podemos aplicar o seletor ao atributo "Class" de qualquer elemento HTML. Se a intenção é aplicar a mesma regra CSS a diferentes elementos, você pode utilizar um Seletor de Classe, segue exemplo:

< p class = "diferente"> Este parágrafo sairá diferente por causa da utilização

A CSS deve começar com um "." (ponto) imediatamente seguido pelo nome da classe, veja:

.diferente { color: black;}

O uso do seletor de classe possibilita estilos diferentes para o mesmo tipo de elemento, exemplo:

p {
margin: 10px;
}
p .destaque {
font-weight: bold;
}
p .discreto {
color: gray;
}

é permitido declar mais de um nome para a classe do mesmo elemento separando os nomes por espaço, exemplo:

< p class "destaque discreto" > Este parágrafo sofrerá estilização de duas regras CSS

Seletor id

O identificador id pode ser aplicado na maioria dos elementos HTML, só que ele deve ser unico em sua página.

< div id="main" >...

A CSS deve ser precedida pelo sinal "#" e imediatamente pelo nome, Veja:

#main {

...

...

...

}
