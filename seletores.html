Seletores css

Um documento html bem estruturado fornece a estrutra a qual seus estilos são aplicados. Para poder estilizar um determinado elemento html utilizando css, você precisa ter uma maneira de selecionar esses elementos. No css a parte de uma regra de estilo que faz isso e chamado de Seletor

Seletor tipo (tags)

É o seletor mais abrangente do css, ele casa com todos os elementos encontrados.
Exemplo:

h1

{Font-size: 20px;}

p

{Font-size: 10px;}

Os estilos acima irão refletir em todos os elementos "p" e "h1" da página

Agrupamento de seletores

Para aplicar uma regra css comum a vários seletores você pode agrupa-los.

Abaixo vemos algumas regras em comum.

h1{ color: red;}
h2{ color: red;}
p{ color: red;}

Aqui eles foram agrupados.

h1, h2, p{ color: red;}

Seletor de classe

Podemos aplicar o seletor ao atributo "class" de qualquer elemento html. Se a intenção é aplicar a mesma regra css a diferentes elementos, você pode utilizar um Seletor de Classe, segue exemplo:

< p> < span class="diferente">Lorem Ipsum is simply dummy text of the printing and typesetting industry. < /p>
<p><span class="diferente">Lorem Ipsum</span> is simply dummy text of the printing and typesetting industry.</p>

//Podemos mudar a cor, fonte ou tamanho da palavra "Lorem Ipsum", utilizando o nome dado na classe dela no css

O css deve começar com um "." (ponto) imediatamente seguido pelo nome da classe, veja:

.diferente { color: blue;}

Lorem Ipsum is simply dummy text of the printing and typesetting industry.

É permitido declarar mais de um nome para a classe do mesmo elemento separando os nomes por espaço, exemplo:

class = "destaque discreto" > Este parágrafo sofrerá estilização de duas regras css

p {
margin: 10px;
}


p .destaque {
font-weight: bold;
}


p .discreto {
color: gray;
}

Seletor id

O identificador id pode ser aplicado na maioria dos elementos html, só que ele deve ser unico em sua página.

id = "main" ...

O css deve ser precedida pelo sinal "#" e imediatamente pelo nome, Veja:

#main {

...

...

...

}
