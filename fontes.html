Famílias de Fontes

Depois da cor, a fonte (letras) é provavelmente a mais fundamental propriedade de uma página. Nesta pagína será apresentado toda a variedade de estilização para fontes oferecida pelas css .

Uma vez que nem todas as fontes existentes estão instaladas nos computadores do usuário ( existem milhares de fontes e a maioria delas não são gratuita), css fornece um mecanismo para compensar esta lacuna. Você define uma lista de fontes para o seu documento a serem usadas na ordem que você definiu (se a primeiraa da lista não estiver disponível no computador do usuário usa-se a segunda e assim por diante) e a última fonte da lista deverá ser uma Fonte genérica dentre uma das cinco disponíveis: #serif, sans-serif, monospace, cursive e fantasy.



A tabela a seguir mostra exemplos de vários tipos de fontes ( talvez o seu navegador não dê suporte para todas elas) e assim você poderá visualizar como o seu naveador renderiza cada uma das cinco fontes genéricas:
